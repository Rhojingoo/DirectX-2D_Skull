## KEY Manual
→방향키 : 오른쪽이동

←방향키 : 왼쪽이동

Z : Dash

X : Attack

C : Jump

A : Skill-1번

S : Skill-2번

Space : Charactor Change

N : Next Scene

P : Collider render Off

O : Collider render On
